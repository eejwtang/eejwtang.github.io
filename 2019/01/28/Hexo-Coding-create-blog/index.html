Hexo+Coding create blog

写在前面：

  • 没有记录就没有发生，人的大脑大概是一台复杂的高级处理器，但一定不是个可靠的信息存储设备，最近学的新知识总是会随着时间慢慢地忘记细节。折腾的过程就是从不同的信息节点上收集信息，加工处理，整合成自己的工作流程。这就要把他抽象整理，记录下来，下次直接拿起来调用。
    • 把自己掌握的知识拿出来给别人讲一遍，能帮助建立知识体系，加深理解。
    • 在互联网的海洋里找一个小角落，记录着他的发生，万一还能帮助上别人。
      基于此，我萌生了自己撘一个Blog的想法。选择目前比较主流的Hexo + Coding Pages（GitHub Pages）的方式
    • 练习写作：我的导师如是说：“研究生读到最后你会发现写代码是最快乐的事情，写论文一定会比写代码痛苦的。”
